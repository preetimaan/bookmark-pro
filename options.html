Bookmark Cleanup

Find duplicate URLs, empty folders, merge same-name folders, similar/subset URLs, and similar folder names.

Backup first. Removed bookmarks cannot be restored. Export your bookmarks (Chrome menu → Bookmarks → Bookmark manager → ⋮ → Export bookmarks) before deleting.

Merge duplicate folders

Folders with the same name under the same parent. Pick one to keep; others will be merged into it.

Similar / subset URLs

URLs where one is a prefix of another (e.g. example.com and example.com/page). Suggest keeping the shortest.

Similar folder names

Folders with the same name anywhere in the tree. Pick one to keep; others will be merged into it.

Broken links

Check all bookmarked URLs for broken links (404, 403, timeout, etc.). This needs permission to access websites.
